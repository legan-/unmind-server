# Code challenge for Unmind - Server

## Tech Stack:

- ES6
- GraphQL
- Node.js
- Express
- npm
- Postgres

## Getting Started

### Installation

To prepare this app for running you have to:

1. Install **node.js**
2. Install and run **Postgres**
3. `git clone git@github.com:legan-/unmind-server.git`
4. `createdb unmind-02082018`
5. `cd unmind-server`
6. `yarn install`

### Running app

`yarn start`
